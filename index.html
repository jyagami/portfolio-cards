Galería de imagenes

Proyecto de galería de imagenes, creado unicamente con CSS, posee una interfaz agradable e interactiva para el usuario. En el encontramos un catalogo de imagenes, puede ser utilizado ya sea para una galería de muestra de productos o un lugar turístico.

Registro completo

En este pequeño proyecto podremos registrar desde productos, inventario, empleados hasta la realización de ventas y generación de facturas, cuenta con una base de datos interactuando con registros en tiempo real
